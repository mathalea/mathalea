Créer un espace

Sur coopmaths.fr/mathalea, vous avez la possibilité d’enregistrer les scores de vos élèves. Pour être en conformité avec la RGPD, nous avons fait le choix de ne conserver aucune données personnelles sur notre site. Nous ne stockerons aucun nom, aucun prénom, aucune adresse électronique, aucune adresse IP …
Le principe est le suivant :
  • Un professeur, un formateur ou un éducateur se rend sur coopmaths.fr/mathalea.html?v=scores.
  • Il clique sur le bouton « Créer un espace ».
  • Il obtient un identifiant avec 3 lettres majuscules au hasard (par exemple VJH) et une adresse qui lui permettra d’accéder aux scores qu’il faudra conserver précieusement.
  • À ces 3 lettres, le professeur devra en ajouter 2 de son choix pour décrire un classe (par exemple VJH6A, VJHI2, ...).
  • Ensuite, pour chaque élève, il devra ajouter 2 ou 3 autres caractères (par exemple VJH6A01, VJH6A02, VJH6AGN2, VJH6AAB1, VJH6AAB2...).
    • Note sur la création des classes

    La création des classes est automatique.
    Une classe est créée à la première utilisation d'un code de cette classe.
    Par exemple, la classe de 6A est créée pour le professeur dont le code est PPP dès qu'un code du type PPP6AEEE est utilisé pour se connecter.

  • Seul le professeur saura à quel élève correspond chaque code. Et si un élève de la 6A oublie son code, le professeur peut en inventer un nouveau (DNY6AZZ) et lui attribuer immédiatement sans avoir besoin de faire quoi que ce soit sur l’ordinateur.
    • Pour enregistrer des scores dans un espace professeur existant, un élève doit donc :

    • Soit se rendre sur coopmaths.fr/mathalea et sélectionner des exercices.
    • Soit cliquer sur un lien menant vers une sélection d'exercices fournie par son professeur.
    • Cliquer sur l'un des boutons qui permet d'utiliser une vue permettant la connexion.
    • Dans cette vue, cliquer sur connexion
    • Une fenêtre modale s'ouvre, il a alors besoin :
      • Du code prof => 3 lettres majuscules, par exemple LOZ.
      • Du code classe => 2 caractères alphanumériques, les lettres sont des majuscules, par exemple 6A.
      • Le code eleve => 2 ou 3 caractères alphanumériques, les lettres sont des majuscules, par exemple 1ET.
    Il renseigne ce code dans la zone de saisie :

    Après avoir cliqué sur Valider, la fenêtre se ferme. Son identifiant est affiché tant qu'il est connecté :

    Il peut se déconnecter à tout moment. S'il change de fenêtre, il doit éventuellement se reconnecter.
    Lorsqu'il clique sur Vérifier les réponses, son score est automatiquement enregistré dans un fichier accessible au professeur.

  • Enfin, le professeur pourra se rendre sur l’adresse qu’il a obtenue au moment de l’inscription pour télécharger :
    • Soit un fichier tableur par classe et par semaine contenant tous les résultats associés aux différents codes à 7 ou 8 caractères.
    • Soit une archive par classe avec tous les résultats de toutes les semaines pour tous les élèves de la classe, contenant un fichier de traitement Scores_MathAlea.ods
    • Soit une archive avec tous les résultats de toutes les semaines pour tous ses élèves quelle que soit la classe, contenant un fichier de traitement Scores_MathAlea.ods
    Il restera à la charge du professeur d’associer les résultats d’un code avec le nom et le prénom de l’élève que seul lui connaît. Le fichier de traitement Scores_MathAlea.ods pourra en faciliter la tâche.
  • Après extraction de l'archive, le fichier de traitement Scores_MathAlea.ods fonctionne ainsi :
    • Tous les fichiers de semaines sont dans le dossier "Fichiers semaines" de l'archive.
    • Le bouton "Mise à jour" :
      • Récupère la liste des "Code Classe" dans la colonne C de l'onglet "Paramètres"
      • Parcourt le dossier "Fichiers semaines" à la recherche des fichiers "Code Classe"-semaine"1 à 56".csv qui existent et dont le nom ne figure pas déjà dans la liste des fichiers ajoutés (colonne A de l'onglet "Données brutes")
      • Copie le contenu des fichiers dans l'onglet "Données brutes"
    Note sur les macros

    Les macros ne sont nécessaires que pour cette mise à jour qui récupère le contenu de tous les fichiers pour les ajouter. Si quelqu'un préfère ne pas exécuter les macros, il peut copier et coller le contenu des fichiers à la main et quand même profiter de la désanonymisation qui se fait à l'aide de formules.
