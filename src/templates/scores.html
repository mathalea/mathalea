Sur coopmaths.fr/mathalea, vous avez la possibilité d’enregistrer les scores de vos élèves. Pour être en conformité avec la RGPD, nous avons fait le choix de ne conserver aucune données personnelles sur notre site. Nous ne stockerons aucun nom, aucun prénom, aucune adresse électronique, aucune adresse IP …
Le principe est le suivant :
  • Un professeur, un formateur ou un éducateur se rend sur coopmaths.fr/mathalea.html?v=scores.
  • Il clique sur le bouton « Créer un espace ».
  • Il obtient un identifiant avec 3 lettres majuscules au hasard (par exemple DNY) et une adresse qui lui permettra d’accéder aux scores qu’il faudra conserver précieusement.
  • À ces 3 lettres, le professeur devra en ajouter 2 de son choix pour décrire un classe (par exemple DNY6A, DNYI21...).
  • Ensuite, pour chaque élève, il devra ajouter 2 autres caractères (par exemple DNY6A01, DNY6A02, DNY6AGN...).
  • Seul le professeur saura à quel élève correspond chaque code. Et si un élève de la 6A oublie son code, le professeur peut en inventer un nouveau (DNY6AZZ) et lui attribuer immédiatement sans avoir besoin de faire quoi que ce soit sur l’ordinateur.
    • Pour enregistrer des scores dans un espace professeur existant, un élève doit donc :

    • Se rendre sur coopmaths.fr/mathalea
    • Cliquer sur connexion
    • Il a alors besoin :
      • Du code prof => 3 lettres majuscules, par exemple STI.
      • Du code classe => 2 caractères alphanumériques, les lettres sont des majuscules, par exemple 6A.
      • Le code eleve => 2 caractères alphanumériques, les lettres sont des majuscules, par exemple 1E.
    Il renseigne ce code dans la zone de saisie :

    Il peut ensuite fermer la fenêtre et choisir des exercices. Son userId est affiché tant qu'il est connecté :

  • Enfin, le professeur pourra se rendre sur l’adresse qu’il a obtenue au moment de l’inscription pour télécharger un fichier tableur par classe et par semaine contenant tous les résultats associés aux différents codes à 7 caractères. Il restera à la charge du professeur d’associer les résultats d’un code avec le nom et le prénom de l’élève que seul lui connait.
